## 🚀 Getting Started

### Prerequisites

- Node.js (LTS version)
- npm or yarn

### Installation & Running

1. Clone the repository:
   ```bash
   git clone git@github.com:MshadDr/benji-pays-ui.git
   cd benji-pays-ui
   ```
2. Copy the example environment file:

   ```bash
   cp .env.example .env
   ```

3. Install the dependencies:

   - Using npm:
     ```bash
     npm install
     ```
   - Or using yarn:
     ```bash
     yarn install
     ```

4. Run the application:
   ```bash
   npm run dev
   ```
